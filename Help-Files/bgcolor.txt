
================================
Welcome to BG-Color Command Help
================================

Command Syntax:
		bgcolor (color name)

Working:
		This Command Helps Us to Change the Foreground Color
		or The Typing Color

Use:
		bgcolor red
		bgcolor blue

Colors Supported:
		Black
		Red
		Green
		Yellow
		Blue
		Purple
		Cyan
		White

Disclamer:
		This Command Only Changes Background Color. Don't Use it
		to Change Foreground Color
